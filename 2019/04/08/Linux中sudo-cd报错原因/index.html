思考

既然mkdir可以使用sudo可以使用，那么说明这个方法是可行的，那么问题的关键就在于 cd 和 mkdir 有什么不同，于是使用type查看。

从截图中可以看出：

  • cd是Linux内建的命令
  • mkdir是一个外部命令，即为一个应用程序
  • 而而sudo仅仅只对应用程序起作用，sudo foo只意味着以root权限运行foo程序

解决办法

  1. 使用sudo -i命令提升用户权限
  2. 使用sudo -s命令打开特殊shell
